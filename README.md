# 退勤記録 Webアプリ v2

Windows 11しかない場合用の、自分だけで使う退勤記録アプリです。

## v2の変更点
- 各記録の右側に「削除」ボタンを追加
- 記録が少ない月でも1件ずつ削除しやすく修正
- 編集画面内の「この記録を削除」も同じ削除処理に統一
- Service Workerのキャッシュ名を v2 に変更し、更新が反映されやすいように修正

## 機能
- 退勤時間をワンタップ記録
- 1ヶ月分を一覧表示
- 記録の日時部分をタップして時刻編集
- 右側の削除ボタンで1件ずつ削除
- 今月分をコピー
- 今月分を削除
- 保存先はブラウザの localStorage

## 使い方
1. index.html / manifest.webmanifest / sw.js をGitHub Pagesなどに上書きアップロード
2. iPhoneのSafariでそのURLを開く
3. 反映されない場合は、Safariで再読み込み。またはホーム画面アイコンを削除して追加し直す

注意: localStorage保存なので、SafariのWebサイトデータ削除やブラウザ初期化をすると記録が消える可能性があります。
